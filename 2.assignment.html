2. Computer-aided design and cutting

Preparation

Assignment 2 is to use a CAD software to build a parametric design of a pressfit construction kit, then cut the design with a laser cutter and document the process and upload our design files. I started by downloading Fusion 360 but I have never used it before so I had to watch some vidoes to be able to make my design on it. I brainstormed some ideas for my design and came up with a design for a jewelrystand to put my earrings and necklaces. I googled some ideas and got inspiration.

Making the design

First we download Git on: https://git-scm.com/downloads. Next we create an account on github.com. Then we create a new repository, make it public and hatch where it says: Initialize this repository with a README and then create our repository. Next we upload our files and save. Finally we press the setting button and scroll down to GitHub Pages and choose master branch in source, after that the website gives you a link to your website. Here is a link to my GitHub page: https://github.com/gudrung8/gudrung8.github.io
